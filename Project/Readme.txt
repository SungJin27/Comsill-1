1. Visual Studio 용 OpenFrameWork Generator를 이용하여 프로젝트를 생성하고
 생성된 프로젝트의 src 폴더에 기존의 ofApp.h, ofApp.cpp, main.cpp를 제거하고 첨부한
 5개의 소스 파일을 복사/붙여 넣기 한다.

2. visual studio에서 솔루션 탐색기를 오른쪽 버튼 클릭하여 추가 -> 기존 항목을 클릭하여
src 폴더에 저장된 5개의 소스 파일을 추가한다.

3.  생성한 프로젝트 폴더의 bin/data 폴더 안에 첨부한 bgm.mp3, Verdana.ttf를 넣는다.

4. Visual Stduio에서 Ctrl + Shitf + B를 통해 프로젝트를 빌드하고,
 Ctrl + F5를 통해 프로그램을 실행한다.

4. 조작법
마우스 : 버튼 클릭

's' : 배경 음악 재생 또는 멈춤

'ESC' : 프로그램 종료
